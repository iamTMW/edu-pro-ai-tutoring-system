Module 2: Adding with Blanks

Adding with blanks helps us find a missing number in an addition problem.

What Adding with Blanks Means

Sometimes we know some numbers but not all. We use a blank (like __ or ?) for the missing number.

We solve it by finding the number that makes the addition true.

Example idea: 4 + __ = 7. The missing number is 3.

How to Solve Blanks

We can solve blanks by:

  • Counting up from the known number
  • Using our fingers to add up
  • Using a number line to hop forward
  • Using objects (like blocks or cubes) to see how many are needed

Tens and Ones Reminder

When numbers have tens and ones, we usually find the missing ones first, then tens.

Why Adding with Blanks Is Important

It helps us:

  • Check our answers
  • Understand how numbers work together
  • Solve puzzles and games
  • Prepare for more difficult addition and subtraction
