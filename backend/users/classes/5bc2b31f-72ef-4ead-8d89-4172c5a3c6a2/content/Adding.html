Module 1: Addition

Addition means putting things together to find out how many we have in total.

What Addition Means

If you have some things and then get more, addition helps us find out the new amount.

We show addition with the + sign.

We show the answer with the = sign.

Example idea: If you have 4 apples and get 3 more, you put them together to get 7 apples.

How to Think About Addition

We can add by:

  • Counting up (start at one number and hop forward)
  • Using our fingers
  • Using a number line
  • Using objects (like blocks, stickers, cubes)

Tens and Ones Reminder

Some numbers have tens (like 20, 30, 40) and ones (like 1, 2, 3).

When we add, we usually add the ones first, then the tens.

Why Addition Is Important

We use addition all the time:

  • Counting toys
  • Adding time
  • Counting money
  • Playing games
  • Sharing snacks
