Module 5: Review of Adding and Subtracting with Blanks

This review covers the key ideas for solving addition and subtraction problems with missing numbers (blanks).

Adding with Blanks: Key Points

Adding with blanks helps us find a missing number in an addition problem.

Sometimes we know some numbers but not all. We use a blank (like __ or ?) for the missing number.

We solve it by finding the number that makes the addition true.

Example: 4 + __ = 7. The missing number is 3.

How to solve:

  • Counting up from the known number
  • Using fingers to add up
  • Using a number line to hop forward
  • Using objects (blocks or cubes) to see how many are needed

Tens and Ones Reminder:

  • Find the missing ones first, then the tens

Why it’s important:

  • Check our answers
  • Understand how numbers work together
  • Solve puzzles and games
  • Prepare for more difficult addition and subtraction

Subtracting with Blanks: Key Points

Subtracting with blanks helps us find the missing number in a subtraction problem.

Sometimes we know some numbers but not all. We use a blank (like __ or ?) for the missing number.

We solve it by finding the number that makes the subtraction true.

Example: __ - 3 = 7. The missing number is 10.

How to solve:

  • Think “How many do I need to start with to subtract and get this number?”
  • Use counting backwards
  • Use objects (blocks, cubes, stickers) to see how many are left
  • Use a number line to hop backward

Tens and Ones Reminder:

  • Find the missing ones first, then the tens
  • Sometimes borrowing is needed to make subtraction work

Why it’s important:

  • Check our answers
  • Understand how subtraction works
  • Solve puzzles and problems
  • Prepare for more advanced math

Tips for Solving Blanks

  • Break problems into smaller steps and solve one part at a time
  • Visualize numbers with objects or drawings
  • Check your answer using the opposite operation (addition ↔ subtraction)
  • Practice with multiple blanks to improve confidence
  • Pay attention to tens and ones to avoid mistakes
