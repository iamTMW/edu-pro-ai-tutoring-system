Module 3: Subtracting with Blanks

Subtracting with blanks helps us find the missing number in a subtraction problem.

What Subtracting with Blanks Means

Sometimes we know some numbers but not all. We use a blank (like __ or ?) for the missing number.

We solve it by finding the number that makes the subtraction true.

Example idea: __ - 3 = 7. The missing number is 10.

How to Solve Subtraction Blanks

We can solve blanks by:

  • Thinking “How many do I need to start with to subtract and get this number?”
  • Using counting backwards
  • Using objects (blocks, cubes, stickers) to see how many are left
  • Using a number line to hop backward

Tens and Ones Reminder

When numbers have tens and ones, we usually find the missing ones first, then tens.

Sometimes we need to “borrow” to make subtraction work.

Why Subtracting with Blanks Is Important

It helps us:

  • Check our answers
  • Understand how subtraction works
  • Solve puzzles and problems
  • Prepare for more advanced math
